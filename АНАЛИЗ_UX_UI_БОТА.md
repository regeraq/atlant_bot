# Полный анализ UX/UI телеграм-бота для аренды автомобилей

## 1. Пользовательские сценарии (User Flow)

### 1.1. Первый запуск и регистрация

#### Триггер: Команда `/start`
**Логика:** 
- Регистрация пользователя в базе данных
- Проверка, является ли пользователь администратором
- Получение минимальной цены из доступных автомобилей (если пользователь не админ)

**Сообщение/Действие бота (обычный пользователь):**
```
👋 Добро пожаловать, {user_name}!

🚗 OLIMP AUTO
Аренда автомобилей с правом выкупа

🎯 Почему выбирают нас:
• 🚙 Широкий выбор автомобилей
• 💰 От {min_price:,} ₽/сутки
• ⚡ Быстрое оформление
• 🛡️ Полная поддержка 24/7

👇 Используйте кнопки меню для навигации.
```

**Кнопки главного меню:**
- 🚗 Каталог автомобилей
- 👤 Мой профиль
- 📞 Контакты
- ℹ️ Помощь

---

#### Триггер: Команда `/help`
**Логика:** Отображение справочной информации

**Сообщение:**
```
📚 Справка

Команды
/start — главное меню
/help — эта справка

Функции
• 🚗 Каталог автомобилей — просмотр доступных машин
• 👤 Мой профиль — информация об активной аренде
• 📞 Контакты — связь с менеджером

Используйте кнопки меню для навигации.
```

---

### 1.2. Просмотр каталога автомобилей

#### Триггер: Кнопка "🚗 Каталог автомобилей" или текст "🚗 Каталог автомобилей"
**Логика:** 
- Регистрация пользователя (если еще не зарегистрирован)
- Получение всех доступных автомобилей
- Проверка, есть ли доступные автомобили

**Сообщение (если есть автомобили):**
```
🚗 КАТАЛОГ АВТОМОБИЛЕЙ

━━━━━━━━━━━━━━━━━━━━━━
📊 Доступно: {len(cars)} автомобилей
━━━━━━━━━━━━━━━━━━━━━━

💡 Выберите автомобиль для просмотра подробной информации
```

**Кнопки:**
- Кнопки автомобилей (до 5 на странице): `🟢 {car_name}\n💰 {price}/день`
- Пагинация: `← Назад` | `{page}/{total_pages}` | `Вперед →`
- 🔄 Обновить
- 🏠 Главное меню

**Сообщение (если нет автомобилей):**
```
Автопарк временно недоступен

Мы обновляем автопарк и добавляем новые автомобили.

Попробуйте позже — мы уведомим, когда появятся новые автомобили.
```

**Кнопки:**
- Обновить
- 🏠 Главное меню

---

#### Триггер: Callback `cars_page:{page}`
**Логика:** Переход на указанную страницу каталога

**Сообщение:** То же, что и при открытии каталога, но с обновленной страницей

---

#### Триггер: Callback `refresh_cars`
**Логика:** Обновление каталога с временной меткой

**Сообщение:**
```
🚗 КАТАЛОГ АВТОМОБИЛЕЙ

━━━━━━━━━━━━━━━━━━━━━━
📊 Доступно: {len(cars)} автомобилей
⏰ Обновлено: {current_time}
━━━━━━━━━━━━━━━━━━━━━━

💡 Выберите автомобиль для просмотра деталей:
```

### 1.3. Детальная информация об автомобиле

#### Триггер: Callback `car_details:{car_id}`
**Логика:** 
- Получение информации об автомобиле по ID
- Проверка доступности
- Получение всех доступных изображений (до 3)
- Отправка фотографий (если есть) или текста

**Сообщение (текст):**
```
🚗 {car['name']}

━━━━━━━━━━━━━━━━━━━━━━
💰 Цена: {price_formatted}/день
📊 Статус: {status_text}
━━━━━━━━━━━━━━━━━━━━━━

Описание
{car['description']}

{если недоступен: Автомобиль временно в аренде}
```

**Кнопки (если доступен):**
- 🚗 Забронировать
- 📞 Связаться с менеджером (url или callback)
- ← Каталог
- 🏠 Главное меню

**Кнопки (если недоступен):**
- ⏰ Уведомить о появлении
- ← Каталог
- 🏠 Главное меню

---

#### Триггер: Callback `back_to_catalog`
**Логика:** Возврат к каталогу автомобилей

**Сообщение:** То же, что и при открытии каталога (страница 0)

---

### 1.4. Бронирование автомобиля

#### Триггер: Callback `book_car:{car_id}`
**Логика:** 
- Получение информации об автомобиле
- Проверка наличия настроенного контакта для бронирования

**Сообщение:**
```
🚗 БРОНИРОВАНИЕ АВТОМОБИЛЯ

━━━━━━━━━━━━━━━━━━━━━━
🚙 {car['name']}
💰 Цена: {price_formatted}/день
━━━━━━━━━━━━━━━━━━━━━━

Следующие шаги
1. Свяжитесь с менеджером
2. Укажите даты аренды
3. Подтвердите бронирование

Менеджер ответит в течение 5 минут
```

**Кнопки (если контакт настроен):**
- Связаться с менеджером (url)
- К автомобилю | Каталог
- Главное меню

**Сообщение (если контакт не настроен):**
```
Контакт для бронирования не настроен

Администратор должен настроить контакт.
Свяжитесь с поддержкой через раздел "Контакты".
```

**Кнопки:**
- ← К автомобилю | Каталог
- Контакты

---

#### Триггер: Callback `car_unavailable`
**Логика:** Автомобиль недоступен

**Сообщение (alert):** "Этот автомобиль сейчас недоступен"

---

#### Триггер: Callback `notify_car:{car_id}`
**Логика:** Подписка на уведомление о появлении автомобиля

**Сообщение (alert):** "Вы будете уведомлены, когда {car['name']} станет доступен"

---

### 1.5. Профиль пользователя

#### Триггер: Кнопка "👤 Мой профиль" или текст "👤 Мой профиль"
**Логика:** 
- Получение активной аренды пользователя
- Проверка наличия активной аренды
- Расчет количества дней аренды
- Расчет следующей даты напоминания (для weekly/monthly)

**Сообщение (если есть активная аренда):**
```
👤 Мой профиль

🚗 Арендуемый автомобиль
{car_name}

💰 Стоимость {price_formatted}/день
📅 Начало аренды {start_date_formatted}
📆 Дней в аренде {days_rented}

⏰ Напоминание об оплате
Время: {reminder_time}
Частота: {type_name}
{если weekly/monthly: 📅 Следующее напоминание: {next_reminder_date}}

Для вопросов свяжитесь с менеджером
```

**Кнопки:**
- 📞 Связаться с менеджером (url или callback)
- 🏠 Главное меню

**Сообщение (если нет активной аренды):**
```
👤 Мой профиль

Статус
Нет активной аренды

Как взять автомобиль в аренду
1. Просмотрите каталог
2. Выберите автомобиль
3. Свяжитесь с менеджером

Быстрое оформление — 5 минут
```

**Кнопки:**
- 🚗 Каталог автомобилей
- 📞 Контакты

---
